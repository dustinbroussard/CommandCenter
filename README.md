# CommandCenter

CommandCenter is a command management PWA for creating, organizing, and fixing commands.

## Features

- Create, edit, delete, and organize commands with tags
- Search, filter by tag, and sort commands
- Command detail modal with quick copy to clipboard
- Import/export commands as JSON (with optional settings)
- AI recommendation and fixing via OpenRouter (optional)
- Theme toggle with system preference fallback
- PWA offline support and install prompt

## Security

- OpenRouter API keys are stored in localStorage; use a low-privilege key and rotate it regularly.
